🔬 Diagnostics

Use these checks to isolate problems at each layer of the stack.

1 — App & API Server

Verifies the C backend is running and responding on port 2016 via the VAPIX reverse proxy.


        


        
        

            
2 — Demo / Mock Data

            
Forces a data reload while Demo Mode is on. Checks that mock JSON files are accessible on the device at /usr/local/packages/fifa_wc/mock/.

            
            

        


        
        

            
3 — Force API Refresh

            
Resets the rate-limit timer and triggers an immediate fetch from the live API (or mock if Demo Mode is on). Check the Live tab after ~5 seconds.

            
            

        


        
        

            
4 — VAPIX Display

            
Sends a test message directly to the C1720/C1710 display. If nothing appears, check device credentials in Config.

            
            

        


        
        

            
5 — Full Status Dump

            
Raw JSON from /status — useful for verifying parsed match data.
